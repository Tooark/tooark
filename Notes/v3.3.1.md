# Release Notes v3.3.1

Data de lançamento: 06/06/2026

## 🆕 Resumo

Versão de manutenção focada na atualização da infraestrutura de testes para xUnit v3, com ajuste de referencias de pacotes e adequação da suite aos analisadores da nova versão. Esta release também padroniza o uso de cancelamento cooperativo nos testes, melhorando responsividade em cenários de timeout/cancelamento. Nao ha breaking changes para os pacotes de runtime.

---

## Modificado

- Migração da referência de testes de `xunit` para `xunit.v3` em `Tooark.Tests`.
- Adequação dos testes de Mediator e DI para uso explícito de `TestContext.Current.CancellationToken` em chamadas assíncronas:
  - `SendAsync(...)`
  - `PublishAsync(...)`
  - `Task.Delay(...)`
- Padronização de estilo nos arquivos de teste afetados para manter consistência de legibilidade.

## Correcoes

- Correção de incompatibilidade de pacote no projeto de testes causada por referência ao ID antigo (`xunit`) com versão da linha v3.
- Correção dos erros de analyzer `xUnit1051`, garantindo passagem do token de cancelamento do contexto do teste em APIs que aceitam `CancellationToken`.

## Pacotes Atualizados

- Atualização da stack de testes para xUnit v3:
  - `xunit.v3` -> `3.2.2`
  - `xunit.runner.visualstudio` -> `3.1.5`
- Atualização do gerenciamento central para refletir os IDs/versões corretos dos pacotes de teste.

## Testes

- Suite de testes validada apos as alterações com sucesso:
  - `3556` testes aprovados
  - `0` falhas
  - `0` ignorados

## Impacto

- Sem breaking changes para consumo dos pacotes Tooark em runtime.
- Melhoria na confiabilidade e cancelamento cooperativo da suite de testes.
- Base de testes alinhada com recomendações e analisadores do xUnit v3.
