# Release Notes v3.3.2

Data de lançamento: 11/06/2026

## 🆕 Resumo

Versão de segurança focada na correção de vulnerabilidades de dependências transitivas. Esta release atualiza `System.Security.Cryptography.Xml` para resolver vulnerabilidades de gravidade alta conhecidas e implementa melhorias no gerenciamento centralizado de pacotes. Não há breaking changes para os pacotes de runtime.

---

## Modificado

- Habilitação da propriedade `CentralPackageTransitivePinningEnabled` no `Directory.Packages.props` para controle centralizado de dependências transitivas.
- Adição explícita de `System.Security.Cryptography.Xml` ao catálogo de pacotes gerenciados centralmente.

## Correcoes

- Correção de vulnerabilidade de gravidade alta em `System.Security.Cryptography.Xml` 8.0.2:
  - Atualização para versão `8.0.3` que resolve as vulnerabilidades conhecidas.
- Garantia de que todas as dependências transitivas sejam gerenciadas e rastreáveis através do Central Package Management.

## Pacotes Atualizados

- `System.Security.Cryptography.Xml` -> `8.0.3`

## Testes

- Suite de testes validada apos as alterações com sucesso:
  - Compatibilidade mantida com todas as versões anteriores
  - Nenhuma regressão identificada

## Impacto

- Sem breaking changes para consumo dos pacotes Tooark em runtime.
- Melhoria significativa na postura de segurança com resolução de vulnerabilidades transitivas.
- Maior controlabilidade e rastreabilidade de dependências em todo o monorepo.
- Alinhamento com melhores práticas de gestão de pacotes em soluções .NET com múltiplos projetos.
