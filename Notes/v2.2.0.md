# Release Notes v2.2.0

Data de lançamento: 05/01/2026

## Mudanças principais

### Novo Pacote: Tooark.Securities

Adicionado novo pacote para segurança em aplicações .NET, fornecendo serviços de **criptografia AES** e **autenticação JWT** com suporte a múltiplos algoritmos.

#### Serviços

| Classe | Descrição |
|--------|-----------|
| `JwtTokenService` | Serviço para criação e validação de tokens JWT |
| `CryptographyService` | Serviço de criptografia/descriptografia AES-256 |

#### Interfaces

| Interface | Descrição |
|-----------|-----------|
| `IJwtTokenService` | Contrato para manipulação de tokens JWT |
| `ICryptographyService` | Contrato para criptografia e descriptografia |

#### DTOs

| Classe | Descrição |
|--------|-----------|
| `JwtTokenDto` | Dados para criação de token (id, login, security) |
| `UserTokenDto` | Resultado da validação do token |

#### Options

| Classe | Descrição |
|--------|-----------|
| `JwtOptions` | Configurações do JWT (algoritmo, chaves, issuer, audience, expiração) |
| `CryptographyOptions` | Configurações de criptografia (algoritmo, secret) |

#### Extensions

| Classe | Descrição |
|--------|-----------|
| `RoleClaimsExtension` | Extensão para converter JwtTokenDto em Claims |
| `TooarkDependencyInjection` | Extensão para registro de serviços no DI |

#### Algoritmos JWT Suportados

- **Simétricos (HMAC):** HS256, HS384, HS512
- **Assimétricos (RSA):** RS256, RS384, RS512, PS256, PS384, PS512
- **Assimétricos (ECDsa):** ES256, ES384, ES512

#### Algoritmos de Criptografia Suportados

- **AES-256-GCM:** Modo GCM (recomendado - com autenticação integrada)
- **AES-256-CBC:** Modo CBC com IV aleatório
- **AES-256-CBC (Legado):** Modo CBCUnsafe com IV zerado (apenas para compatibilidade)

### Impacto

- Novo pacote disponível: `Tooark.Securities`
- Sem breaking changes em pacotes existentes

## Configuração

Para utilizar o novo pacote de segurança:

```bash
dotnet add package Tooark.Securities
```

Configure no `appsettings.json`:

```json
{
  "Jwt": {
    "Algorithm": "HS256",
    "Secret": "sua-chave-secreta-com-pelo-menos-32-caracteres",
    "Issuer": "sua-aplicacao",
    "Audience": "seus-clientes",
    "Expires": 60
  },
  "Cryptography": {
    "Algorithm": "GCM",
    "Secret": "sua-chave-secreta-de-32-caracteres"
  }
}
```

Registre os serviços no `Program.cs`:

```csharp
builder.Services.AddTooarkSecurity(builder.Configuration);
```
