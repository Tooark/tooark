# Release Notes v2.1.9

Data de lançamento: 29/10/2025

## Mudanças principais

### Segurança

- Aplicado patch de segurança do .NET (SDK/Runtime) utilizado pelos projetos da solução, alinhado às atualizações de segurança mais recentes da Microsoft.
- Recomenda-se realizar um rebuild completo para garantir que o ambiente utilize a versão corrigida.

### Impacto

- Sem mudanças de API pública.
- Não são esperadas alterações de comportamento em tempo de execução.

### Ações recomendadas

- Atualize o SDK/Runtime local para a mesma versão do pipeline/ambiente de execução, quando aplicável.
- Limpe artefatos e restaure dependências antes do próximo build para garantir a aplicação do patch.

## Outros ajustes

- Documentação de notas atualizada para refletir esta versão.
